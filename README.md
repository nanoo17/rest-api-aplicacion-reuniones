## Instrucciones

### Crear un archivo .env y rellenar los campos con las variables de entorno requeridas por el servidor

```bash
# ./.env

# Puerto donde correra el servidor
PORT=
# Cadena de conexion de mongo
MONGO_URI=
# Firma del jwt
SECRET_JWT_SEED=
```

### Iniciar en modo desarrollador

```
npm run dev
```
